# 美团黄黑编辑风 · 设计规范

> 目标：**一眼是美团，但版式的设计感远强于官网。**
> 手法 = 美团品牌色 DNA + 编辑/杂志式排版（巨字、编号、留白、撞色、拼贴）。

## 1. 色板与角色

| 令牌 | 值 | 角色 |
|---|---|---|
| `--mt-yellow` | `#FFD100` | 主黄。**只做三件事**：关键词高亮块、撞色底、数据英雄块 |
| `--mt-yellow-deep` | `#FFC300` | 压边 / 描线的深黄 |
| `--mt-yellow-soft` | `#FFF3C4` | 极浅黄，弱高亮（少用） |
| `--mt-ink` | `#141414` | 墨黑：巨字、墨块反白底、2px 主线 |
| `--mt-cream` | `#FBF9F1` | 暖白页底（浅色为主的默认底） |
| `--mt-gray` / `--mt-gray-2` | `#8C8B84` / `#B9B7AD` | 注释灰 / 深底上的次级文字 |
| `--mt-line` | `#E6E3D8` | 1px 细分割线 |

**禁忌**：不做渐变、不加阴影光晕、不引入第三个彩色。黄色一旦铺满整页就失去「高亮」语义——铺满只留给章节页/结束页当撞色节奏。

## 2. 字体与层级

| 用途 | 字体 | 字重 | 说明 |
|---|---|---|---|
| 西文巨字 / 编号 / 数据 | Archivo | 900 | 顶天立地的编辑感来源 |
| 中文标题 | Noto Sans SC（思源黑） | 900 Heavy | 行高 `.96~1.1`，字距 `-.01em` |
| 正文 | Noto Sans SC | 400/500 | 行高 `1.6~1.8`，**不拉字距** |
| 小型大写标签 | Archivo | 800 | `letter-spacing:.16~.2em`，大写 |
| 点缀手写 | Caveat | 700 | **仅英文短句**（Eat Better…/powered by AI），每页 ≤1 处 |

字号阶梯（1280×720 画布）：巨字 `132/104/74`，区块标题 `60/30/26`，数据 `92/74/56`，正文 `15~18`，标签 `12~13`。

## 3. 版面骨架（编辑感的核心）

- **页眉**：左 logo 徽标（黄底方块「美团」+ MEITUAN），右 页面主题；下压一条 **2px 墨线**。
- **页脚**：左信息 / 中行动号召（可选）/ 右页码 `01 / 24`。小型大写、灰色。
- **主区**：左对齐锚定，大面积留白；焦点元素（巨字/数据/图）占据视觉重心。
- **网格**：外边距 64px；多栏用等宽网格 + 细线/直角分隔（`.ed-card` 用 2px 墨边直角，不圆角）。

## 4. 母题元素

- **黄块高亮** `.hl-mark`：给关键词打底黄，跨行用 `box-decoration-break:clone`。
- **墨块反白** `.ink-block`：深色词块，反白字，做与黄块的对位。
- **四角星芒**：内联 SVG 星形，墨黑或黄色，每页 0~1 个，置于角落呼吸区。
- **黑白照 + 偏移块** `.photo`：图片 `grayscale(1)`，背后 `::before` 垫一块黄色/墨色，制造拼贴层次。
- **箭头母题** `.chev`：大标题后跟 `▶▶▶` 实心三角（黑），呼应「THE STORY」式导向感，每页 ≤1 组。
- **编号照片网格** `.sb-grid` / `.sb-cell`：`(01)(02)(03)` + 粗体小标题 + 说明 + 黑白照，做流程/分镜/案例走查；常配满黄画布 `.canvas-y`。
- **人物卡** `.person`：黑白头像 + 黑色名条（名字 + 黄色英文职务）；搭配黑色大数字卡 `.card-stat`（黄色巨数）做团队/嘉宾页。
- **巨型描边字母** `.outline`（`.cjk` 用中文）：`-webkit-text-stroke` 的空心巨字（如「美」），作满黄封面的背景线稿母题，溢出画布边缘。
- **满黄 / 满墨画布** `.canvas-y` / `.canvas-ink`：整页撞色，用作封面/章节/流程/结束等重音页。
- **细节家具（全局）**：`.slide::after` 四角定位角标 + `.rule` 端帽 + `.kicker::after` 尾刻度，印刷编辑感。颜色走 `--mark`（浅/黄底墨色，深底加 `.slide-dark` 转奶白），1.5px 细线、半透明，克制不抢戏。

## 5. 节奏（整册编排）

浅色为主、深/黄撞色作重音：
```
封面(暖白) → 目录(暖白) → 章节01(黄底) → 内容(暖白·黑白卡) →
数据(暖白·黄英雄块) → 金句(墨底) → 案例(暖白·图文) → 时间轴(暖白) → 结束(黄底)
```
原则：**连续 2~3 页浅色后，插一页黄底或墨底撞色**，制造记忆点，避免通篇平。

## 6. 每页自检清单

- [ ] 只有一个视觉焦点？其余是否压成了小字注释？
- [ ] 黄色是否 ≤2 处、且承担的是高亮/撞色/数据角色？
- [ ] 中文标题是 Heavy(900)、行高够紧？
- [ ] 页眉 2px 墨线、页脚页码都在？
- [ ] 星芒 / 手写点缀是否克制（各 ≤1）？
- [ ] 配图是否已转黑白并套了偏移块？

## 7. 灵感来源（方向参考）

- 黄+黑+白 编辑网站（大字 + 手写体 + 黑白照拼贴）—— 与美团色几乎无缝
- 巨型字品牌手册（`01/02` 编号、整屏撞色、极简注释）—— 结构与气势
- 编辑式品牌手册（小型大写标签、页码、克制网格）—— 版面家具的精致度
